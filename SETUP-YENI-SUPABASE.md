# Yeni Supabase projesi – SQL kurulumu

1. Supabase Dashboard → **SQL Editor** → **New Query**
2. **`setup-yeni-supabase.sql`** dosyasının tamamını kopyala, yapıştır, **Run**
3. Bitti. Başvuru formu, giriş ve aday paneli için yeterli.

İsteğe bağlı (müşteri/ilan, site ayarları, storage vb.): `add-customer-role-enum-only.sql` → `add-customer-role-complete.sql`, `create-site-settings-table.sql`, `create-hero-sliders-table.sql`, `STORAGE-KURULUM.md` içindeki adımlar.
